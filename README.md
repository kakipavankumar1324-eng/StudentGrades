# Student Grades Management System

A simple Java application designed to manage and calculate student grades.

## 🚀 Features
* Input and store student names and their respective scores.
* Calculate average, highest, and lowest grades.
* Display a formatted summary of student performance.

## 🛠️ Prerequisites
* **Java Development Kit (JDK)** 8 or higher installed.

## 💻 How to Run
1. Clone the repository or download the source files.
2. Open your terminal or command prompt.
3. Compile the Java file:
   ```bash
   javac StudentGrades.java
